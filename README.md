# norruva-coming-soon

A simple coming soon homepage with beta signup.

## Deployment to Vercel

1. Commit and push the code to GitHub.
2. Go to [Vercel](https://vercel.com) and sign in.
3. Click "New Project" and import this repository.
4. Vercel will automatically detect it as a static site and deploy it.

The site will be live at a URL provided by Vercel.
